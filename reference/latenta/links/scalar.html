Scalar links¶

Functions that map a scalar variable (with * dimensions) to another scalar variable (with * + # dimensions, where # can be empty) Each component allows an x variable which solely consists of batch dimensions, often only one (as per torch convention) Each component will often add a b variable, which has * + # dimensions. Some extra dimensions can be added to this variable as well, e.g.: spline functions will add a knot dimension.

class latenta.links.scalar.Linear(x: latenta.variables.variable.Variable, *args, output=None, **kwargs)[source]¶

Bases: latenta.links.scalar.scalar.Scalar

Components:

  • «x»: The input.

  • «a»: The slope (also known as weight). Denotes how much the output changes with each addition of «x». If True, the component will be initialized as a Latent variable, using the output difference distribution as prior.

  • «b»: The intercept (also known as bias). Denotes the output when «x» = 0. If True, the component will be initialized as a Latent variable, using a Normal distribution with both \(\mu\) and \(\sigma\) as parameters.
