class latenta.variables.Observation(value, p, *args, **kwargs)[source]¶

Bases: latenta.variables.random.RandomMixin, latenta.variables.fixed.Fixed

An observed random variable, i.e. the value of this variable is uncertain (=follows a distribution \(p\)) but observed (=we have a sample from this distribution). The parameters of the variables’ distribution \(p\) may depend on other observed, latent, fixed or parameter variables. using likelihood

class latenta.variables.Latent(p, q=None, definition=None, initial=None, initial_quantiles=None, **kwargs)[source]¶

Bases: latenta.variables.random.VariationalMixin, latenta.variables.random.RandomMixin, latenta.variables.variable.Variable

A latent random variable i.e. the actual value of this variable is uncertain and unobserved. The variable follows a prior distribution \(p\), which can have parameters that may be fixed and/or depend on other variables. The posterior distribution of a latent is approximated using a variational distribution \(q\). \(q\) is often but not necessarily a Normal distribution, with some additional transformations so that its support matches that of \(p\). \(q\) has variational parameters that need to be estimated, which are by default done directly through parameter variables, but which can also be estimated from the data using an amortization function.
